# Analytics With Python

![Frame 48](https://github.com/celesteg21/Analytics-with-Python/assets/112521035/f66b487b-4654-4455-8a47-333309c414f3)


In this repository, you will find the code blocks used for data analysis with Python.

----------
The idea of this V1 of the repository is to maintain the simplicity of the code to perform basic analyses with Python, using a small dataset and utilizing the matplotlib and seaborn libraries for creating simple visualizations. You can take code snippets and adapt them to your needs, gradually enhancing your code for more in-depth analyses.

In V1, we will upload the analysis performed in Tableau to have a visual option and understand the data behavior.

In a V2, we will create more complex code where we consolidate various datasets || connect to the Spotify API to build predictive models for songs, understand users based on their emotions, and generate a recommendation system.

----------
La idea de esta V1 de repositorio es mantener la simplicidad del código para poder realizar análisis básicos con python, desde un dataset pequeño y utilizar las librerías de matplotlib y seaborn para realizar gráficos simples. 
Puedes tomar los fragmentos de código y adaptarlos a tus necesidades y complejizar tu código para obtener mejores análisis.

En la V1' vamos a estar subiendo el análisis pero realizado en Tableau, para tener la opción visual y entender el comportamiento de los datos.

En una V2 vamos a realizar un código más complejo, donde unifiquemos varios datasets || nos conectemos a la API de Spotify para realizar modelos de predicción de canciones, entender a los usuarios en base a sus emociones y generar un recomendador.
